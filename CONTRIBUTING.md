# Contributing to Termux-CLI

## Getting Started

1. Fork and clone the repository
2. Run `./scripts/setup.sh`
3. Create a feature branch
4. Make changes and run tests
5. Submit a pull request

## Code Style

- Follow PEP 8
- Use type hints
- Write docstrings
- Run `black .` and `ruff check .` before committing

## Testing

```bash
./scripts/test.sh
```
